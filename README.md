# article_activation_function

В данном репозитории представлен текст студенческой статьи и код, на основе результатов которого была написана практическая часть статьи. 

Тема статьи: "Сравнительный анализ функций активации в задаче классификации изображений в MobileNetV2".
Проводилось сравнение 7 нелинейных функций активации: ReLU, ReLU6, PReLU, ELU, MISH, Swish, Hard-Swish, 
на модели MobileNetV2 на датасете Cifar-10. Сравнение проводилось по 2м метрикам: accuracy, inference time, на основе
метода взвешенной суммы.

В результате лучшей по комбинации 2х метрик получилась функция Mish. Теоретические ожидания о времени работы функций
активаци не подтвердились. Был сделан вывод о необходимости эмпирически проверять функции активации в целевых условиях, 
так как на скорость работы модели могут влиять реализация функций в библиотеках и оптимизация на GPU.
